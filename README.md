# GravityIssue  
  
A mod that adds landing platforms for rocket boosters (*ASOG* and *Jacklyn LPV-1*)  
  
  
## FEATURES  

* A remote controlled droneship with move ability like ***SpaceX Launch Vehicles*** mod.  
* The mass balance prevents the barge from rocking during the landing of the rocket boosters.  
* Statics (requires Kerbal Konstructs).  
  
  
## DEPENDENCIES
* _B9PartSwitch_   
* _ModuleManager_
* _Kerbal Konstructs_ (for static objects)
  
## RECOMMENDED MODS
* _Booster Guidance_
* _FMRS_ (Flight Manager for Reusable Stages)
* _Tundra Exploration_ (for Falcon 9 rocket)
* _Big Blue_ (for New Glenn rocket)
* _Trajectories_
